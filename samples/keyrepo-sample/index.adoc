### Embedded per message encryption key sample

In this example, an encryption key reference is embedded in the record payload.
The encryption key is generated by the key repository. Two records with the same record's key embbed the same encryption
key reference. The key repository is a simple in memory repository. The encryption keys are stored encrypted (by a master key).


#### Running the sample

[source,indent=0]
....
cd samples/kafka-encryption-generatedkey-sample
docker-compose up # on windows and OSX, you need to adjust
sh generateMasterKey.sh
sh runSamples.sh
....

##### On OSX and Windows

[source,indent=0]
....
docker run --rm -p 2181:2181 -p 3030:3030 -p 8081-8083:8081-8083 -p 9581-9585:9581-9585 -p 9092:9092 -e ADV_HOST=<YOUR MACHINE IP ADDRESS> landoop/fast-data-dev:cp4.0.0
....

##### On linux

[source,indent=0]
....
docker run --rm --net=host -e ADV_HOST=localhost landoop/fast-data-dev:cp4.0.0
....
